# Blue-DOS

Blue-DOS: A Bluetooth Denial of Service Tool

### How it works

Keep sending pair requests to available Bluetooth devices.

### Installation

1. Clone the repository:

   ```sh
   git clone https://github.com/Owbird/Blue-Dos.git
   cd Blue-Dos
   ```

2. Install dependencies:

   ```sh
   go mod download
   ```

3. Run the project:
   ```sh
   go run .
   ```
